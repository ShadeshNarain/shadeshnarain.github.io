FAQ

Quick answers for new runners.

Which class should I start with?

If you’re new: Beginner Endurance or the Free Group Run. If you already run: Aerobic Base + Drills & Dynamic Warmup.

What’s in “Drills & Dynamic Warmup”?

Mobility + activation + running drills (form, cadence, coordination). Great before speed/tempo days.

Do I need to book the free run?

No booking needed for the free 6:00 AM group run—just show up at the meeting point.
